## Nix Release Notes

The Nix release cycle is calendar-based as follows: Nix has a release cycle of roughly 6 weeks. Notable changes and additions are announced in the release notes for each version. The supported Nix versions are: The latest release The version used in the stable NixOS release, which is announced in the NixOS release notes. Bugfixes and security issues are backported to every supported version. Patch releases are published as needed.
