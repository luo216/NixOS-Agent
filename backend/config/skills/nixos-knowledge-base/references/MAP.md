# Nix Ecosystem Map

This skill stores local documentation converted into Markdown branches.

## Branches

- Flakes book: `references/flakes-book/INDEX.md`
- Nix manual: `references/nix-manual/INDEX.md`
- Nixpkgs manual: `references/nixpkgs-manual/INDEX.md`
- NixOS options/modules: `references/options-modules/INDEX.md`
- Community navigation: `references/community/INDEX.md`
- Wiki navigation: `references/wiki/INDEX.md`

## Start Here

- For NixOS + Flakes setup, migration, and practical workflow questions, start with `references/flakes-book/INDEX.md`.
- For Nix command/reference questions, start with `references/nix-manual/INDEX.md`.
- For Nixpkgs packaging/library questions, start with `references/nixpkgs-manual/INDEX.md`.
- Nixpkgs chapters are further split under `references/nixpkgs-manual/chapters/` and `references/nixpkgs-manual/release-notes/`.
- For NixOS option/module questions, start with `references/options-modules/INDEX.md`.
- Use `references/community/INDEX.md` for community resource discovery only, not as the default factual source.
- Use `references/wiki/INDEX.md` only as a supplemental navigation layer when the official manuals do not answer the question directly.
