# programs.atop.atopacctService

## programs.atop.atopacctService.enable

Whether to enable the atopacct service which manages process accounting. This allows Atop to gather data about processes that disappeared in between two refresh intervals. Type: boolean Default: true Declared by: <nixpkgs/nixos/modules/programs/atop.nix>
