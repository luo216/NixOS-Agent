# documentation.dev

## documentation.dev.enable

Whether to install documentation targeted at developers. This includes man pages targeted at developers if documentation.man.enable is set (this also includes “devman” outputs). This includes info pages targeted at developers if documentation.info.enable is set (this also includes “devinfo” outputs). This includes other pages targeted at developers if documentation.doc.enable is set (this also includes “devdoc” outputs). Type: boolean Default: false Declared by: <nixpkgs/nixos/modules/misc/documentation.nix>
