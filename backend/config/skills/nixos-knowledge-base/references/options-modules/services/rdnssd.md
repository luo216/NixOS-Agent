# services.rdnssd

## services.rdnssd.enable

Whether to enable the RDNSS daemon (rdnssd), which configures DNS servers in /etc/resolv.conf from RDNSS advertisements sent by IPv6 routers. Type: boolean Default: false Declared by: <nixpkgs/nixos/modules/services/networking/rdnssd.nix>
