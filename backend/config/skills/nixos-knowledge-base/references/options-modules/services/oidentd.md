# services.oidentd

## services.oidentd.enable

Whether to enable ‘oidentd’, an implementation of the Ident protocol (RFC 1413). It allows remote systems to identify the name of the user associated with a TCP connection. Type: boolean Default: false Declared by: <nixpkgs/nixos/modules/services/networking/oidentd.nix>
