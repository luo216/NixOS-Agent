# services.syncplay

## services.syncplay.enable

If enabled, start the Syncplay server. Type: boolean Default: false Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.package

The syncplay-nogui package to use. Type: package Default: pkgs.syncplay-nogui Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.certDir

TLS certificates directory to use for encryption. See https://github.com/Syncplay/syncplay/wiki/TLS-support. Type: null or absolute path Default: null Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.chat

Chat with users in the same room. Type: boolean Default: true Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.extraArgs

Additional arguments to be passed to the service. Type: list of string Default: [ ] Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.interfaceIpv4

The IP address to bind to for IPv4. Leaving it empty defaults to using all. Type: string Default: "" Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.interfaceIpv6

The IP address to bind to for IPv6. Leaving it empty defaults to using all. Type: string Default: "" Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.ipv4Only

Listen only on IPv4 when strting the server. Type: boolean Default: false Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.ipv6Only

Listen only on IPv6 when strting the server. Type: boolean Default: false Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.isolateRooms

Enable room isolation. Type: boolean Default: false Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.maxChatMessageLength

Maximum number of characters in a chat message. Type: unsigned integer, meaning >=0 Default: 150 Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.maxUsernameLength

Maximum number of characters in a username. Type: unsigned integer, meaning >=0 Default: 16 Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.motd

Text to display when users join. The motd will be readable in the nix store and the processlist. If this is not intended use motdFile instead. Will be overriden by services.syncplay.motdFile. Type: null or string Default: null Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.motdFile

Path to text to display when users join. Will override services.syncplay.motd. Type: null or string Default: if services.syncplay.motd != null then (builtins.toFile "motd" services.syncplay.motd) else null Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.passwordFile

Path to the file that contains the server password. If null, the server doesn’t require a password. Type: null or absolute path Default: null Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.permanentRooms

List of rooms that will be listed even if the room is empty. Will be overriden by services.syncplay.permanentRoomsFile. Type: list of string Default: [ ] Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.permanentRoomsFile

File with list of rooms that will be listed even if the room is empty, newline delimited. Will override services.syncplay.permanentRooms. Type: null or string Default: if services.syncplay.permanentRooms != [ ] then (builtins.toFile "perm" (builtins.concatStringsSep "\n" services.syncplay.permanentRooms)) else null Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.port

TCP port to bind to. Type: 16 bit unsigned integer; between 0 and 65535 (both inclusive) Default: 8999 Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.ready

Check readiness of users. Type: boolean Default: true Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.roomsDBFile

Path to SQLite database file to store room states. Relative to the working directory provided by systemd. Type: null or string Default: null Example: "rooms.db" Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.salt

Salt to allow room operator passwords generated by this server instance to still work when the server is restarted. The salt will be readable in the nix store and the processlist. If this is not intended use saltFile instead. Mutually exclusive with services.syncplay.saltFile. Type: null or string Default: null Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.saltFile

Path to the file that contains the server salt. This allows room operator passwords generated by this server instance to still work when the server is restarted. null, the server doesn’t load the salt from a file. Mutually exclusive with services.syncplay.salt. Type: null or absolute path Default: null Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.statsDBFile

Path to SQLite database file to store stats. Relative to the working directory provided by systemd. Type: null or string Default: null Example: "stats.db" Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>

## services.syncplay.useACMEHost

If set, use NixOS-generated ACME certificate with the specified name for TLS. Note that it requires security.acme to be setup, e.g., credentials provided if using DNS-01 validation. Type: null or string Default: null Example: "syncplay.example.com" Declared by: <nixpkgs/nixos/modules/services/networking/syncplay.nix>
