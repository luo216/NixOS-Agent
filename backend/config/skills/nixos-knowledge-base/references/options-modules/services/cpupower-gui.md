# services.cpupower-gui

## services.cpupower-gui.enable

Enables dbus/systemd service needed by cpupower-gui. These services are responsible for retrieving and modifying cpu power saving settings. Type: boolean Default: false Example: true Declared by: <nixpkgs/nixos/modules/services/desktops/cpupower-gui.nix>
