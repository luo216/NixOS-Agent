# services.waagent.settings.Logs

## services.waagent.settings.Logs.Verbose

If you set this option, log verbosity is boosted. Waagent logs to /var/log/waagent.log and uses the system logrotate functionality to rotate logs. Type: boolean Default: false Declared by: <nixpkgs/nixos/modules/virtualisation/waagent.nix>
