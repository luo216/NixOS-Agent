# services.synergy.server.tls

## services.synergy.server.tls.enable

Whether TLS encryption should be used. Using this requires a TLS certificate that can be generated by starting the Synergy GUI once and entering a valid product key. Type: boolean Default: false Declared by: <nixpkgs/nixos/modules/services/misc/synergy.nix>

## services.synergy.server.tls.cert

The TLS certificate to use for encryption. Type: null or string Default: null Example: "~/.synergy/SSL/Synergy.pem" Declared by: <nixpkgs/nixos/modules/services/misc/synergy.nix>
