# services.rpcbind

## services.rpcbind.enable

Whether to enable rpcbind, an ONC RPC directory service notably used by NFS and NIS, and which can be queried using the rpcinfo(1) command. rpcbind is a replacement for portmap. Type: boolean Default: false Declared by: <nixpkgs/nixos/modules/services/networking/rpcbind.nix>
