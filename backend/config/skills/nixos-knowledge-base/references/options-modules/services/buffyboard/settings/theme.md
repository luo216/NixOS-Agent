# services.buffyboard.settings.theme

## services.buffyboard.settings.theme.default

Selects the default theme on boot. Can be changed at runtime to the alternative theme. Type: string or one of <null>, “adwaita-dark”, “breezy-dark”, “breezy-light”, “nord-dark”, “nord-light”, “pmos-dark”, “pmos-light” Default: null Declared by: <nixpkgs/nixos/modules/services/hardware/buffyboard.nix>
