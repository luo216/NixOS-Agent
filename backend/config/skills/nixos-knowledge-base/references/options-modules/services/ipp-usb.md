# services.ipp-usb

## services.ipp-usb.enable

Whether to enable ipp-usb, a daemon to turn an USB printer/scanner supporting IPP everywhere (aka AirPrint, WSD, AirScan) into a locally accessible network printer/scanner. Type: boolean Default: false Example: true Declared by: <nixpkgs/nixos/modules/services/printing/ipp-usb.nix>
