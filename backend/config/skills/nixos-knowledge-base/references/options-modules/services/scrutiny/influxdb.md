# services.scrutiny.influxdb

## services.scrutiny.influxdb.enable

Enables InfluxDB on the host system using the services.influxdb2 NixOS module with default options. If you already have InfluxDB configured, or wish to connect to an external InfluxDB instance, disable this option. Type: boolean Default: true Declared by: <nixpkgs/nixos/modules/services/monitoring/scrutiny.nix>
