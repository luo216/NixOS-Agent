# services.evremap.settings.remap.*

## services.evremap.settings.remap.*.input

The key sequence that should be remapped. You can get a list of keys by running evremap list-keys. Type: list of key ID prefixed with BTN_ or KEY_ Declared by: <nixpkgs/nixos/modules/services/misc/evremap.nix>

## services.evremap.settings.remap.*.output

The key sequence that should be output when the input sequence is entered. You can get a list of keys by running evremap list-keys. Type: list of key ID prefixed with BTN_ or KEY_ Declared by: <nixpkgs/nixos/modules/services/misc/evremap.nix>
