# services.evremap.settings.dual_role.*

## services.evremap.settings.dual_role.*.hold

The key sequence that should be output when the input key is held. You can get a list of keys by running evremap list-keys. Type: list of key ID prefixed with BTN_ or KEY_ Declared by: <nixpkgs/nixos/modules/services/misc/evremap.nix>

## services.evremap.settings.dual_role.*.input

The key that should be remapped. You can get a list of keys by running evremap list-keys. Type: key ID prefixed with BTN_ or KEY_ Declared by: <nixpkgs/nixos/modules/services/misc/evremap.nix>

## services.evremap.settings.dual_role.*.tap
