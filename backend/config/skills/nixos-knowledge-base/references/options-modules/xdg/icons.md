# xdg.icons

## xdg.icons.enable

Whether to install files to support the XDG Icon Theme specification. Type: boolean Default: true Declared by: <nixpkgs/nixos/modules/config/xdg/icons.nix>

## xdg.icons.fallbackCursorThemes

Names of the fallback cursor themes, in order of preference, to be used when no other icon source can be found. Set to [] to disable the fallback entirely. Type: list of string Default: [ ] Declared by: <nixpkgs/nixos/modules/config/xdg/icons.nix>
