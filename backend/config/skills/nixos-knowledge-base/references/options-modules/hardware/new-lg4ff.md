# hardware.new-lg4ff

## hardware.new-lg4ff.enable

Enables improved Linux module drivers for Logitech driving wheels. This will replace the existing in-kernel hid-logitech modules. Works most notably on the Logitech G25, G27, G29 and Driving Force (GT). Type: boolean Default: false Declared by: <nixpkgs/nixos/modules/hardware/new-lg4ff.nix>
