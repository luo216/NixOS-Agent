# hardware.tuxedo-drivers

## hardware.tuxedo-drivers.enable

Whether to enable The tuxedo-drivers driver enables access to the following on TUXEDO notebooks: Driver for Fn-keys SysFS control of brightness/color/mode for most TUXEDO keyboards Hardware I/O driver for TUXEDO Control Center For more inforation it is best to check at the source code description: https://gitlab.com/tuxedocomputers/development/packages/tuxedo-drivers . Type: boolean Default: false Example: true Declared by: <nixpkgs/nixos/modules/hardware/tuxedo-drivers.nix>
