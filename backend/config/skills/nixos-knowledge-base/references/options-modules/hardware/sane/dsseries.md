# hardware.sane.dsseries

## hardware.sane.dsseries.enable

When enabled, will automatically register the “dsseries” SANE backend. This supports the Brother DSmobile scanner series, including the DS-620, DS-720D, DS-820W, and DS-920DW scanners. Type: boolean Default: false Example: true Declared by: <nixpkgs/nixos/modules/services/hardware/sane_extra_backends/dsseries.nix>
