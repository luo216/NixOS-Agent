# hardware.usb-modeswitch

## hardware.usb-modeswitch.enable

Enable this option to support certain USB WLAN and WWAN adapters. These network adapters initial present themselves as Flash Drives containing their drivers. This option enables automatic switching to the networking mode. Type: boolean Default: false Declared by: <nixpkgs/nixos/modules/hardware/usb-modeswitch.nix>
