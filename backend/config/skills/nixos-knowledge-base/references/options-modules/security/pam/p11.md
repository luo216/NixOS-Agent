# security.pam.p11

## security.pam.p11.enable

Enables P11 PAM (pam_p11) module. If set, users can log in with SSH keys and PKCS#11 tokens. More information can be found here. Type: boolean Default: false Declared by: <nixpkgs/nixos/modules/security/pam.nix>

## security.pam.p11.control

This option sets pam “control”. If you want to have multi factor authentication, use “required”. If you want to use the PKCS#11 device instead of the regular password, use “sufficient”. Read pam.conf(5) for better understanding of this option. Type: one of “required”, “requisite”, “sufficient”, “optional” Default: "sufficient" Declared by: <nixpkgs/nixos/modules/security/pam.nix>
