# boot.loader

## boot.loader.timeout

Timeout (in seconds) until loader boots the default menu item. Use null if the loader menu should be displayed indefinitely. Type: null or signed integer Default: 5 Declared by: <nixpkgs/nixos/modules/system/boot/loader/loader.nix>
