# environment.stub-ld

## environment.stub-ld.enable

Install a stub ELF loader to print an informative error message in the event that a user attempts to run an ELF binary not compiled for NixOS. Type: boolean Default: true Example: false Declared by: <nixpkgs/nixos/modules/config/stub-ld.nix>
